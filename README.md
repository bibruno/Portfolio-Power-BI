## Girassol RH

![Gif que exemplifica o funcionando do Dashboard RH](https://i.imgur.com/7FLV0gf.gif[/img])

#### Principais elementos da tabela fato:
- cpf_func
- data_nascimento
- genreo_func
- status (DAX)
- IDADE_ATUAL (DAX)
- FAIXA_ETARIA (DAX)
- idNivel

#### Principais elementos das tabelas dimensão:
- Nivel (dCargos)
- tipo_func (dTipoFuncionario)
- mes (DAX - dCalendario)

## Sales Dashboard
![Gif que exemplifica o funcionando do Sales Dashboard](https://i.imgur.com/TmvcXJy.gif[/img])
#### Principais elementos da tabela fato:
- PRODUCT ID
- QUANTITY
- SALE TYPE
- PAYMENT MODE
- DISCOUNT
- DATE

#### Principais elementos DAX:
Total Sales, Total Profit, Total Product, Top Category, Profit%

#### Principais elementos das tabelas dimensão:
- BUYING PRIZE
- CATEGORY
- PRODUCT
- PRODUCT ID
- PRODUCT ID
